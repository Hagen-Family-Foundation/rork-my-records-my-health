Prioritized App Store screenshot plan

Use these screenshots in this order. The first five tell the strongest story: emergency readiness, organized medical records, controlled sharing, wallet-card backup, and secure access.

Direct links below are app routes. In the web preview they open the right screen when the app is running. On a phone, use the tab or menu instructions if a route link does not open directly.

Recommended final App Store order

  1. Emergency View — strongest opening shot because it sells the life-saving purpose immediately.
  2. Records Dashboard — proves the app organizes the user’s medical profile in one place.
  3. Share Screen — shows controlled exports and privacy-first sharing.
  4. Wallet Card — highlights the screenshot/print backup feature.
  5. Security Settings — addresses PIN, Face ID, biometric unlock, and privacy concerns.
  6. Language Settings — include if you want a sixth screenshot focused on accessibility and multilingual support.
  7. Critical Documents — include if you want to show photo storage for medical documents.
  8. Search — include if you want to show speed and convenience.
