# Add document viewing, sharing, emailing, and privacy controls

## Features
- [x] Tap any saved document to open a clear action menu.
- [x] View image documents inside the app before deciding what to do next.
- [x] Open PDF and other medical files with the phone’s trusted file viewer.
- [x] Share the actual saved file through the phone’s share options, including Mail, Gmail, AirDrop, Files, and provider apps.
- [x] Offer an email-friendly option for sending a document to a doctor when the user chooses.
- [x] Keep the existing written health record export separate from actual file sharing so users stay in control.
- [x] Add a privacy reminder before sharing an actual medical file.
- [x] Add an on/off setting so the user can choose whether that sharing reminder appears.
- [x] Record sharing actions in the activity history without exposing private medical details in the log.

## Design
- [x] Use a calm, medical-record style action sheet with simple choices: View, Share, Email, and Cancel.
- [x] Show a strong but respectful privacy reminder before a file leaves the app.
- [x] Make document cards clearer by showing whether the item is an image, PDF, or other file.
- [x] Keep the flow discreet and private, with no automatic sending or silent upload.

## Pages / Screens
- [x] **Documents**: Tapping a saved document opens the new viewing and sharing choices.
- [x] **Document Preview**: Images can be reviewed in a focused full-screen view with sharing options.
- [x] **Share**: The existing health record sharing area remains available for summary exports.
- [x] **Settings**: A new privacy reminder toggle lets the user decide whether file-sharing warnings stay on.
